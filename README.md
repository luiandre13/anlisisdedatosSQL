# 📊 Curso de Análisis de Datos con SQL

## 🎯 Propósito

El propósito de este curso es proporcionar a los participantes las habilidades esenciales para realizar análisis de datos utilizando SQL. Al finalizar el curso, los estudiantes estarán capacitados para extraer información valiosa de bases de datos relacionales y contribuir significativamente al proceso de toma de decisiones basado en datos.

## 🧠 Temas del Curso

1. **Introducción a las bases de datos y SQL**  
   - Qué es una base de datos relacional  
   - Arquitectura básica de bases de datos  
   - Historia y propósito del lenguaje SQL

2. **Fundamentos de SQL**  
   - Sintaxis básica  
   - Tipos de datos  
   - Operadores lógicos y aritméticos

3. **Consultas en SQL**  
   - `SELECT`, `FROM`, `WHERE`  
   - Filtros condicionales  
   - Consultas básicas y avanzadas

4. **Manipulación de Datos**  
   - `INSERT`, `UPDATE`, `DELETE`  
   - Consideraciones de integridad y transacciones

5. **Ordenamiento, Agrupación y Filtro de Resultados**  
   - `ORDER BY`, `GROUP BY`, `HAVING`  
   - Filtrado después del agrupamiento

6. **Funciones de Agregación**  
   - `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`  
   - Combinación con `GROUP BY`

7. **Joins: Combinar Datos de Múltiples Tablas**  
   - `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `FULL JOIN`  
   - Casos prácticos y relaciones entre tablas

8. **Subconsultas**  
   - Subconsultas en `SELECT`, `WHERE`, y `FROM`  
   - Subconsultas correlacionadas

9. **Cláusulas Avanzadas**  
   - `CASE`, `COALESCE`, `NULLIF`, `EXISTS`, `IN`, `ALL`, `ANY`  
   - `LIMIT`, `OFFSET`, `WITH` (CTEs)
